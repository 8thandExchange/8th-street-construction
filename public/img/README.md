# Static images

Place static assets here (logo files, favicon, og-image, etc.).

Project gallery images live in **Supabase Storage** (`project-images` bucket), not here.

Suggested files to add:
- `favicon.ico`
- `icon.png` (512×512 — Next.js auto-generates favicon variants)
- `apple-icon.png` (180×180)
- `og-default.jpg` (1200×630 — default Open Graph image)
- `logo.svg` (if you have a vector mark)
